# mattjamesmartin.github.io
Source code for [mattjmartin.com](https://mattjmartin.com/). My name is Matt Martin, and I am a postdoctoral researcher at the [Comparative Constitutions Project](https://comparativeconstitutionsproject.org/). In August 2026, I will start as an Assistant Professor in the Department of Political Science at the University of South Carolina.
